# DVR
**行车记录仪系统**

- 开发平台：Qt5.14.2
- 操作系统：Windows10
- 语言：C++，C
- 使用框架：FFPMEG

**文件夹说明**：

| 目录 |                             说明                             |
| :--: | :----------------------------------------------------------: |
| bin  |                          可执行文件                          |
| doc  | 项目相关文档（架构图，业务流程图，表结构，类图，部分业务逻辑图） |
| src  |                             源码                             |
